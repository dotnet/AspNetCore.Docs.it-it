> [!IMPORTANT]
> **Versioni di anteprima di ASP.NET Core con il Servizio app di Azure**
>
> Le versioni di anteprima di ASP.NET Core non sono distribuite al Servizio app di Azure per impostazione predefinita. Per ospitare un'applicazione che usa una versione di anteprima di ASP.NET Core, vedere [Distribuire la versione di anteprima di ASP.NET Core in Servizio app di Azure](xref:host-and-deploy/azure-apps/index#deploy-aspnet-core-preview-release-to-azure-app-service).
